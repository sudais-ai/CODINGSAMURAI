📖 Quotes Scraper Project


🔹 About

A simple Python project that scrapes random quotes from Quotes to Scrape.
It shows quotes with authors in the terminal and also saves them to a text file.

🔹 Features

Collects quotes from the website

Shows them neatly in terminal

Saves top 8 quotes into my_quotes.txt

Beginner-friendly and easy to use

🔹 Technologies

Python 3

Requests (fetch website data)

BeautifulSoup4 (parse HTML)

Time (smooth output)

🔹 How To Run
git clone https://github.com/sudais-ai/CODINGSAMURAI.git
cd CODINGSAMURAI
pip install requests beautifulsoup4
python scraper.py


Output will be shown in terminal and saved in my_quotes.txt.
